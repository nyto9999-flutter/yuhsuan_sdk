# Yuhsuan Sdk

Yuhsuan sdk

## Installation 💻

**❗ In order to start using Yuhsuan Sdk you must have the [Dart SDK][dart_install_link] installed on your machine.**

Add `yuhsuan_sdk` to your `pubspec.yaml`:

```yaml
dependencies:
  yuhsuan_sdk:
```

Install it:

```sh
dart pub get
```
